CAMPUS FLOW - COMPLETE INSTALLATION GUIDE
==========================================

QUICK SETUP (5 Minutes)
------------------------

1. INSTALL XAMPP
   - Download and install XAMPP from https://www.apachefriends.org/
   - Make sure Apache and MySQL are installed

2. START XAMPP SERVICES
   - Open XAMPP Control Panel
   - Start Apache
   - Start MySQL

3. COPY PROJECT FILES
   - Copy all project files to: C:\xampp\htdocs\campusflow\
   - Make sure folder structure is maintained:
     * api/ folder (with all PHP files)
     * js/ folder (with all JavaScript files)
     * All HTML files in root
     * config.php in root
     * database.sql in root

4. CREATE DATABASE
   - Open browser and go to: http://localhost/phpmyadmin
   - Click on "New" to create a new database
   - Name it: campus_flow
   - Click "Create"
   - Select the database from left sidebar
   - Click on "Import" tab
   - Click "Choose File" and select database.sql
   - Click "Go" to import

5. VERIFY SETUP
   - Open browser and go to: http://localhost/campusflow/setup_check.php
   - This will verify all files and database connections
   - Fix any issues shown

6. ACCESS APPLICATION
   - Open browser and go to: http://localhost/campusflow/
   - Register a new account or login with:
     Username: admin
     Password: password

TROUBLESHOOTING
---------------

Problem: "Connection failed" error
Solution: Make sure MySQL is running in XAMPP

Problem: "404 Not Found" error
Solution: Check that files are in correct folder and Apache is running

Problem: "Database doesn't exist" error
Solution: Create database 'campus_flow' and import database.sql

Problem: Pages show but data doesn't load
Solution: Check browser console (F12) for JavaScript errors

Problem: Can't login after registration
Solution: Check that sessions are enabled in PHP (usually enabled by default)

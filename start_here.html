Step 1: Start XAMPP

1. Open XAMPP Control Panel

2. Click "Start" for Apache (wait until it turns GREEN)

3. Click "Start" for MySQL (wait until it turns GREEN)

4. Both must show "Running" status

Step 2: Verify Apache is Running

1. Open browser and go to: http://localhost/

2. You should see the XAMPP welcome page

3. If you see this, Apache is working! ‚úÖ

Step 3: Check Project Location

Your project MUST be in:

C:\xampp\htdocs\campusflow\

If it's not there:

  • Copy your project folder to C:\xampp\htdocs\
  • Rename it to campusflow

Step 4: Setup Database

1. Open: http://localhost/phpmyadmin

2. Click "New" ‚Üí Database name: campus_flow ‚Üí "Create"

3. Select campus_flow database

4. Click "Import" tab

5. Choose database.sql file ‚Üí Click "Go"

Step 5: Access Your Application

‚úÖ Correct URLs (use these):

  • http://localhost/campusflow/

‚ùå Wrong URLs (don't use these):
